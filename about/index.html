このサイトは？

Web制作、レポート作成、スケジュール管理など、あらゆる作業用途での利用を想定し作られたWebアプリ集です。
すべてブラウザ上で動作します。今後もアプリを増やしていきますのでお楽しみに。

動作環境

必須条件

  • JavaScriptが有効化されていること。
    当サイトのWebアプリはJavaScriptで実装しています。
  • Cookie類似技術の使用が許可されていること。
    当サイトではCookieは使用していませんが、一部のアプリでCookieの類似技術であるLocal Storageを使用しています。
    処理はすべてローカル環境上で行われますので、Local Storageの内容がサーバなど外部に送信されることはありません。
  • HTML5に対応していること。

動作確認済環境
※以下は正常に動作することを確認できたブラウザのみ記載しています。記載していないブラウザでも正常に動作する可能性があります。

  • Google Chrome バージョン95.0以降 (Windows / macOS / iOS / iPadOS / Android)
  • Firefox バージョン94.0以降 (Windows)
  • Microsoft Edge バージョン95.0以降 (Windows)
  • Safari 最新版 (macOS / iOS / iPadOS)
